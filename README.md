## 📌 About

MailCraft is a free, fully client-side web tool that helps students and professionals generate personalized internship cold emails instantly. No backend, no API keys, no cost — just fill in your details and get 3 unique email variations ready to send.

Built as part of the **Digital Heroes Trial Task**.

---

## ✨ Features

- 📧 **3 Email Variations** — Classic Professional, Story-Driven, Bold & Confident
- 🎯 **Smart Personalization** — Emails generated using your actual name, college, skills, company and role
- 💼 **LinkedIn DM Generator** — 300-char connection message auto-generated
- 💡 **Subject Line Suggestions** — 3 subject line options per generation
- 📄 **PDF Export** — Download any email as a PDF
- ⬇️ **TXT Download** — Save email as plain text file
- 📋 **One-Click Copy** — Copy full email with confetti animation ✨
- 🕓 **Email History** — Last 5 generated emails saved locally
- 🌙 **Dark Glassmorphism UI** — Premium animated dark theme
- 📱 **Mobile Responsive** — Works on all screen sizes

## 💡 How to Use

1. Enter your **personal details** — name, college, degree, year, skills
2. Enter **target details** — company name, role, tone, email type
3. Click **✨ Generate Emails**
4. Get **3 unique email variations** instantly
5. **Copy, Download PDF, or Download TXT**
6. Use the **LinkedIn DM** for connection requests
7. Pick from **3 subject line suggestions**
